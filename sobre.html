Introdução

O cenário do trabalho proposto é que nós fomos contratados pela IBM para desenvolver um provador de fórmulas proposicionais, onde ficamos responsáveis pelo analisador léxico, sintático e o provador de tautologia.


Analisador Léxico

Verifica se os simbolos da proposição estão corretos.

Analisador Sintático

Identifica se sintaxe do input atende aos requisitos de uma fórmula bem formulada(FBF ou WFF).

Provador de Tautologia

Testa e identifica quando houver uma tautologia.

Metodologia

Durante o estudo prévio para execução do trabalho notamos que existem diversas formas de desenvolver uma solução, e existem maneiras mais simples de se resolver, como a utilização de bibliotecas já prontas. A maneira que optamos foi desenvolver nosso próprio programa, a ponto de fixar e entender melhor a lógica proposicional.

Python: Library TTG(Table-Truth-Generator)

Resultados

Nossa ferramenta consegue fazer a Análise Léxica e Sintática porém, como não conseguimos realizar operações mais complexas, não foi possível também fazer um Provador Tautológico, mas, que seria bem simples apenas verificando se todos os itens da array são 1's(uns) ou True



Conclusão

Nossa ferramenta foi capaz de realizar a análise Léxica e Sintática porém, com a complexidade do desenvolvimento em javascript, não foi possível realizar operações maiores do que as com 2 termos, pensamos em outras formas de abordar o problema, como arrays com True e False no lugar de 1 e 0 para apenas usar os operadores em javascript, e funções para condicional e bicondicional, e de maneira recursiva, gerar uma array de resultados a cada operação(uma letra com um operador, ou uma letra um operador e uma letra) e ir comparando com as demais arrays resultados já criadas, porém não conseguimos implementar. O Javascript é muito mais interessante para o desenvolvimento da lógica matemática, computacional e proposicional, já o Python é ideal para realizar o projeto sem mais complexidade, apenas focado no resultado das operações e no trabalho com grandes volumes de dados.
